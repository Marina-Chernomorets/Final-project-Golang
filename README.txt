# Final-project-Golang
Этот код - мини-веб-приложение на Go (Golang), которое предоставляет API для добавления математических выражений, их вычисления и получения информации о них.

1. Expression - структура, представляющая математическое выражение, его статус (ожидание или завершено), результат, время создания и завершения.

2. main функция инициализирует обработчики запросов (handlers) для маршрутов "/add" (добавление выражения), "/expressions" (получение всех выражений) и "/expression" (получение выражения по ID). Приложение запускается на порту 8080.

3. calculateExpression функция вычисляет математическое выражение, переданное в виде строки.

4. addExpressionHandler - обработчик для добавления нового математического выражения. Он принимает POST запрос с выражением, создает новый объект Expression, добавляет его в список выражений и асинхронно вычисляет результат. В ответ отправляется ID добавленного выражения.

5. getExpressionsHandler - обработчик для получения списка всех выражений. Он возвращает текстовое представление всех выражений.

6. getExpressionByIDHandler - обработчик для получения информации о выражении по его ID. Он принимает GET запрос с параметром ID, ищет соответствующее выражение и возвращает информацию о нем.

7. testAPI функция используется для тестирования API. Она добавляет выражение "2 + 2" с помощью POST запроса и затем получает список всех выражений через GET запрос.

Когда вы добавляете новое выражение, оно сохраняется в памяти приложения, а затем асинхронно вычисляется его результат. При получении списка всех выражений или конкретного выражения по ID, возвращается информация о статусе, результате и времени создания/завершения.

Этот код демонстрирует простую реализацию веб-приложения на Go для работы с математическими выражениями через API.


Чтобы запустить этот код, вам нужно выполнить следующие шаги:

1. Установите Go (Golang) на свой компьютер, если у вас его еще нет. Можете сделать это с помощью официальной инструкции на сайте: https://golang.org/doc/install.

2. Скопируйте данный код в файл с расширением ".go", например, "main.go".

3. Откройте терминал (командную строку) и перейдите в директорию, где находится ваш файл "main.go".

4. Запустите приложение, выполнив команду go run main.go. После этого приложение должно начать работу и поднимет сервер на порту 8080.

5. После запуска приложения вы можете выполнить тестирование API, добавив выражение "2 + 2" с помощью функции testAPI(). Вам будет выведена информация о добавленном выражении и полученном списке всех выражений.

6. Теперь вы можете отправлять запросы к вашему API через HTTP-запросы на http://localhost:8080/add для добавления новых выражений, http://localhost:8080/expressions для получения списка всех выражений и http://localhost:8080/expression?id=ID для получения информации о конкретном выражении по его ID.
